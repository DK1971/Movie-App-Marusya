# Финальная работа - курс Vue 3 (Skillbox - 2025)
## Над чем предстоит работать
Вы разработаете онлайн-платформу «VK Маруся», которая позволит пользователям находить и оценивать фильмы. 

Ваша задача — создать функциональность поиска фильмов, настроить возможность их оценки и обеспечить простоту и удобство использования приложения.

В процессе работы над платформой «VK Маруся» вы примените все полученные в результате прохождения профессии навыки: 
- HTML, CSS (вёрстка), 
- JavaScript, 
- TypeScript,
- Vue.js (Pinia, Vue Router).
